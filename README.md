===========
浏览器要求：
使用火狐或chrome最新版本（本项目没做兼容测试）

===========
环境要求：

node.js 6.0 以上版本

===========
目录说名：
BNG_Project
/dist 是打包后文件生产目录，也是部署时使用的目录
/src  是开发时使用的目录

===========
安装：
进入项目目录执行 npm install 安装各种依赖

===========
开发模式

进入项目，在根目录下执行
node dev
启动后访问：localhost:8080/

===========
部署模式

进入项目，在根目录下执行
npm run deploy

将js html scss 等文件打包的dist目录下，形成静态文件
